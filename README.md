# bubu-lifestyle-enterprise-erp
A production-ready Omnichannel ERP &amp; Multi-outlet POS for a footwear brand. Features aggregated inventory sync, inter-outlet transfers, PIN-protected security, and a custom loyalty engine. Includes automated logistics (Steadfast), fraud detection (BDCourier), and server-side tracking (Meta CAPI). Built with React, Supabase, and AI.
